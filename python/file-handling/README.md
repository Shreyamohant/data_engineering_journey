# Python File Handling Project

## Description
This project demonstrates practical file handling in Python including:
- Writing to files
- Reading files
- Cleaning data
- Logging execution
- Error handling

## Features
- Creates and reads text files
- Removes empty lines and extra spaces
- Generates cleaned output
- Records execution time
- Handles file not found errors

## Files
- file_project.py → Main program  
- data.txt → Raw input data  
- clean_data.txt → Cleaned output  
- log.txt → Execution logs  

## How to Run
python file_project.py

## Learning Outcome
Understood real-world file processing and data cleaning workflow.
